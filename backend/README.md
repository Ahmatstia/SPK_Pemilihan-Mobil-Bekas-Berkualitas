# MOORA DSS Backend API

Backend API untuk Sistem Pendukung Keputusan Pemilihan Mobil Bekas menggunakan Metode MOORA.

## Fitur

- CRUD untuk Kriteria
- CRUD untuk Alternatif (Mobil Bekas)
- Perhitungan MOORA lengkap
- Analisis akurasi dengan Confusion Matrix
- Data inisial dari jurnal penelitian

## Instalasi

1. Install dependencies:

```bash
npm install
```
